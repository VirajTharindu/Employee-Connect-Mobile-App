# ✨ Key Features

Employee Connect is a comprehensive management system tailored for administrative and social service workflows.

## 📊 Advanced Reporting Engine
The system generates granular reports based on various socio-economic factors:

| Category | Description |
|---|---|
| **Social Aids** | Tracking Samurdhi, Aswasuma, Elderly (Wedihiti), Disabled (Abhadhitha), and other local aid programs. |
| **Employment** | Categorized statistics for Government, Semi-Government, Private, Forces, and Self-Employed individuals. |
| **Education** | Specific tracking for school-age students and higher education levels among adults. |
| **Demographics** | Distribution analysis based on Religion, Ethnicity, and Age Groups (including legal age tracking). |

## 🖨 PDF Generation & Data Portability
- **Direct PDF Export**: Filter any report and generate a professional PDF document for printing or digital sharing.
- **Database Management**: Integrated tools for database sharing and importing, ensuring data availability across devices.

## 🔐 Security & Licensing
- **Encrypted Licensing**: Access to the system is restricted via a 256-bit encrypted license activation system.
- **Secure Storage**: Sensitive license keys and user data are handled using `flutter_secure_storage` and `shared_preferences`.

## 🌐 Responsive Web Container
The application features a specialized `Max-Width` builder for Web and Desktop deployments, providing a focused, mobile-like experience even on ultra-wide displays.
